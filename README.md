# Unified Digital Governance & Social Intelligence Platform

A minimalistic, professional web platform demo showcasing digital governance infrastructure. Single-page application with mock data—no backend required.

## Quick Start

Open `index.html` in a browser, or run a local server:

```bash
npx serve .
# or
python3 -m http.server 8000
```

Then visit `http://localhost:8000` (or the port shown).

## Sections

1. **Dashboard Overview** — Executive metrics (parliamentary questions, committees, attendance, youth issues, citizen submissions, narrative alerts)
2. **Narrative Monitoring** — Trending topics, sentiment, misinformation alerts, suggested responses
3. **MP Performance Transparency** — Parliamentary questions by category, topic distribution chart, timeline
4. **Content Engine** — Speech-to-content transformation (Reel, Twitter, Carousel, Youth Simplified)
5. **Youth & Skill Development Hub** — Policy explainers, Q&A preview, student poll
6. **Citizen-to-Parliament Portal** — Issue submission form, status tracking, recurring issues analytics
7. **X Analytics Intelligence** — Engagement metrics, content performance, best posting times

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- No frameworks or build step
- Fully responsive
